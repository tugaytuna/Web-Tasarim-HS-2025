Ders İçeriği

Html bir işaretleme dilidir. Biz eğitimlerimizin tamamında html etiketlerini ve html özellikleri sık sık kullandık. Html dünyadaki web sitelerinin %99'unda kullanılan web siteleri geliştirmek için olmazsa olmaz bir teknolojidir.

CSS web sitelerinin tasarım ihtiyaçlarını karşılamak için oluşturulmuş bir tasarım dilidir. HTML ile kod yazdıktan sonra css kullanarak web sitemizi kullanıcılara daha güzel görünen arayüzler tasarlamak için kullanıyoruz. Bu sayede Css dili oldukça önemli bir yer kaplıyor.

Bootstrap bir tasarım kütüphanesidir. Ücretsiz ve açık kaynak kodlu olması bizim kullanmamız açısından önemli bir fırsat sunuyor. Web sitelerindeki arayüzleri daha iyi bir hale getirmek için Bootstrap'i kullanıyoruz. Sıfırdan kendi doküman dosyalarını okuyarak kurmayı ve düzenlemeyi öğrendik.

Javascript programlama dili web projelerine işlevler katmamıza olanak sağlıyor. Güçlü altyapısı sayesinde sadece web projelerinde değil aynı zamanda yapay zeka, veritabanı uygulamaları ve diğer konularda da oldukça gelişmiş çözümler üretmemize olanak sağlıyor.
